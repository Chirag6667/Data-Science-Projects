# Software Developer Salary EDA 💰

## Overview
Exploratory Data Analysis on 40,000 software developer
salary records across 10 countries.

## Dataset
- Source: Kaggle — Software Developer Salary Prediction
- Rows: 40,000 | Columns: 7
- Features: experience, country, education,
  languages, frameworks, company_size
- Target: salary_usd

## Key Findings
1. USA pays 2x more than India
2. Experience is strongest salary predictor
3. Education level has minimal salary impact
4. Company size doesn't significantly affect salary
5. Multi-language developers earn premium

## Tech Stack
- Python, Pandas, Matplotlib, Seaborn

## How to Run
1. Download dataset from Kaggle
2. Open notebook in Kaggle or Jupyter
3. Run all cells
